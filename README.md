# Booking System

سیستم رزرواسیون - یک سیستم مدیریت رزرو و نوبت‌دهی

## ویژگی‌ها

- مدیریت کاربران
- سیستم رزرواسیون
- مدیریت نوبت‌ها
- داشبورد مدیریتی

## نصب و راه‌اندازی

```bash
# کلون کردن repository
git clone https://github.com/emadansari96/Booking-System.git

# ورود به پوشه پروژه
cd Booking-System

# نصب وابستگی‌ها
npm install

# اجرای پروژه
npm start
```

## تکنولوژی‌های استفاده شده

- React
- Node.js
- Express
- MongoDB
